# Welcome to Remirror

Remirror is a **React toolkit** for building **cross-platform** text editors, based on [ProseMirror](https://prosemirror.net). It provides a powerful, extensible foundation for creating rich text editing experiences in your React applications.

## What is Remirror?

Remirror is designed to make building sophisticated text editors straightforward and enjoyable. Instead of wrestling with ProseMirror's low-level APIs, you work with React components and hooks that feel natural and familiar.

```tsx
import React from 'react';
import { BoldExtension, ItalicExtension, UnderlineExtension } from 'remirror/extensions';
import { Remirror, useRemirror, OnChangeJSON } from '@remirror/react';

const extensions = () => [new BoldExtension(), new ItalicExtension(), new UnderlineExtension()];

const Editor = ({ onChange }) => {
  const { manager, state } = useRemirror({
    extensions,
    content: '<p>Hi <strong>Friend</strong></p>',
    stringHandler: 'html',
    selection: 'end',
  });

  return (
    <Remirror manager={manager} initialContent={state}>
      <OnChangeJSON onChange={onChange} />
    </Remirror>
  );
};
```

With just a few lines of code, your editor supports bold, italic, and underline formatting with proper keyboard shortcuts, markdown input rules, and HTML paste handling.

## Why Remirror?

ProseMirror is an incredibly powerful editing framework, but it has a steep learning curve. Remirror bridges this gap by:

- **Encapsulating complexity**: Extensions bundle together schemas, commands, keymaps, input rules, and paste rules
- **Providing React bindings**: Hooks and components that integrate seamlessly with your React apps
- **Offering ready-to-use editors**: WYSIWYG, Markdown, and Social editors you can use immediately
- **Enabling customization**: Start with pre-built editors or compose your own from individual extensions

## Core Concepts

### Extensions

Everything in Remirror is an extension. Extensions manage how content is displayed, what commands are available, and how the editor responds to user input.

```tsx
import { BoldExtension, ItalicExtension, HeadingExtension } from 'remirror/extensions';

const extensions = () => [
  new BoldExtension(),
  new ItalicExtension(),
  new HeadingExtension(),
];
```

There are three types of extensions:

- **NodeExtension**: For block-level content (paragraphs, headings, lists)
- **MarkExtension**: For inline formatting (bold, italic, links)
- **PlainExtension**: For behavior that doesn't render in the DOM (history, collaboration)

### Manager & State

The **RemirrorManager** controls what your editor can do. It combines your chosen extensions and manages the editor's lifecycle.

The **EditorState** represents the current content and selection. Remirror handles state management for you, whether you need a controlled or uncontrolled editor.

```tsx
const { manager, state } = useRemirror({
  extensions,
  content: '<p>Initial content</p>',
  stringHandler: 'html',
});
```

### Commands & Helpers

Extensions provide commands that modify the editor content and helpers that query the current state:

```tsx
import { useCommands, useActive } from '@remirror/react';

function BoldButton() {
  const commands = useCommands();
  const active = useActive();
  
  return (
    <button
      onClick={() => commands.toggleBold()}
      disabled={!commands.toggleBold.enabled()}
      style={{ fontWeight: active.bold() ? 'bold' : 'normal' }}
    >
      Bold
    </button>
  );
}
```
