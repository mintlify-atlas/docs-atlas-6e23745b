## Social Editor

<Note>
  The `<SocialEditor>` provides a state-of-the-art editor with mention and tag support.
</Note>

### Installation

```bash
npm install @remirror/react-editors
```

### Usage

```tsx
import { SocialEditor } from '@remirror/react-editors/social';

function App() {
  return <SocialEditor />;
}
```

### Features

- Mention support with @ syntax
- Hashtag support with # syntax
- Emoji support
- Rich text formatting
- Mobile-friendly interface
