List extensions provide structured list support with nesting, indentation, and task management.

## BulletListExtension

Adds unordered (bullet) list support.

### Installation

```typescript
import { BulletListExtension } from 'remirror/extensions';

const extensions = () => [new BulletListExtension()];
```

### Options

<ParamField path="enableSpine" type="boolean" default="false">
  Enable visual spine for nested lists with collapsible functionality
  
  ```typescript
  new BulletListExtension({ enableSpine: true })
  ```
</ParamField>

### Commands

#### `toggleBulletList()`

Toggle bullet list at the current position.

```typescript
commands.toggleBulletList();
```

### Keyboard Shortcuts

- `Mod-Shift-8` - Toggle bullet list (Ctrl/Cmd + Shift + 8)

### Input Rules

Markdown-style patterns:
- `* ` - Creates bullet list
- `- ` - Creates bullet list
- `+ ` - Creates bullet list

### Schema

```typescript
bulletList {
  listItem+ // One or more list items
}
```

## OrderedListExtension

Adds numbered (ordered) list support.

### Installation

```typescript
import { OrderedListExtension } from 'remirror/extensions';

const extensions = () => [new OrderedListExtension()];
```

### Commands

#### `toggleOrderedList()`

Toggle ordered list at the current position.

```typescript
commands.toggleOrderedList();
```

### Keyboard Shortcuts

- `Mod-Shift-7` - Toggle ordered list (Ctrl/Cmd + Shift + 7)

### Input Rules

Markdown-style patterns:
- `1. ` - Creates ordered list starting at 1
- `2. ` - Creates ordered list starting at 2
- `123. ` - Creates ordered list starting at 123

### Schema

```typescript
orderedList {
  attrs: {
    order: number // Starting number (default: 1)
  }
  listItem+ // One or more list items
}
```

### Starting Number

The `order` attribute controls the starting number:

```typescript
// Create a list starting at 5
commands.toggleOrderedList({ order: 5 });
```

## TaskListExtension

Adds checkable task list support.

### Installation

```typescript
import { TaskListExtension } from 'remirror/extensions';

const extensions = () => [new TaskListExtension()];
```

### Commands

#### `toggleTaskList()`

Toggle task list at the current position.

```typescript
commands.toggleTaskList();
```

#### `toggleTaskListItem()`

Toggle the checked state of a task list item.

```typescript
commands.toggleTaskListItem();
```

### Input Rules

Markdown-style patterns:
- `[ ] ` - Creates unchecked task
- `[x] ` - Creates checked task
- `[X] ` - Creates checked task

### Schema

```typescript
taskList {
  taskListItem+ // One or more task items
}

taskListItem {
  attrs: {
    checked: boolean // Checkbox state
  }
  paragraph // Task content
}
```

## ListItemExtension

Defines list item nodes. Automatically included by list extensions.

### Installation

```typescript
import { ListItemExtension } from 'remirror/extensions';

// Usually not needed - included automatically
const extensions = () => [new ListItemExtension()];
```

### Options

<ParamField path="enableCollapsible" type="boolean" default="false">
  Enable collapsible list items
</ParamField>

## List Commands

Additional list manipulation commands:

### Indentation

#### `indentList()`

Increase list item indentation (nest deeper).

```typescript
commands.indentList();
```

**Keyboard Shortcut:** `Tab`

#### `dedentList()`

Decrease list item indentation (nest shallower).

```typescript
commands.dedentList();
```

**Keyboard Shortcuts:** 
- `Shift-Tab`
- `Mod-[`

### Wrapping

#### `wrapInList(type)`

Wrap selected content in a list.

```typescript
commands.wrapInList('bulletList');
commands.wrapInList('orderedList');
```

#### `liftListItem()`

Lift list item out of parent list.

```typescript
commands.liftListItem();
```

#### `sinkListItem()`

Sink list item into previous list item.

```typescript
commands.sinkListItem();
```

### Splitting

#### `splitListItem()`

Split current list item (Enter key behavior).

```typescript
commands.splitListItem();
```

**Keyboard Shortcut:** `Enter`

## Nested Lists

Lists can be nested to create hierarchical structures:

```markdown
* Parent item
  * Nested item 1
  * Nested item 2
    * Deeply nested
* Another parent
```

```typescript
// Programmatically create nested lists
commands.toggleBulletList();
commands.insertText('Parent item');
commands.indentList(); // Nest the next item
commands.insertText('Nested item');
```

## Usage Example

```tsx
import { Remirror, useRemirror } from '@remirror/react';
import {
  BulletListExtension,
  OrderedListExtension,
  TaskListExtension,
  ListItemExtension,
} from 'remirror/extensions';

function ListEditor() {
  const { manager, state } = useRemirror({
    extensions: () => [
      new BulletListExtension({ enableSpine: true }),
      new OrderedListExtension(),
      new TaskListExtension(),
      // ListItemExtension is included automatically
    ],
    content: `
      <ul>
        <li>Bullet item 1</li>
        <li>Bullet item 2</li>
      </ul>
      <ol>
        <li>Numbered item 1</li>
        <li>Numbered item 2</li>
      </ol>
    `,
  });

  return (
    <div>
      <div className="toolbar">
        <button onClick={() => manager.commands.toggleBulletList()}>
          Bullet List
        </button>
        <button onClick={() => manager.commands.toggleOrderedList()}>
          Numbered List
        </button>
        <button onClick={() => manager.commands.toggleTaskList()}>
          Task List
        </button>
        <button onClick={() => manager.commands.indentList()}>
          Indent
        </button>
        <button onClick={() => manager.commands.dedentList()}>
          Dedent
        </button>
      </div>
      <Remirror manager={manager} initialContent={state} />
    </div>
  );
}
```

## Keyboard Navigation

- `Enter` - Split list item / Create new item
- `Tab` - Indent list item (nest)
- `Shift-Tab` - Dedent list item (unnest)
- `Backspace` (at start) - Lift out of list
- `Mod-Shift-7` - Toggle ordered list
- `Mod-Shift-8` - Toggle bullet list

## Markdown Support

All list extensions support markdown syntax:

```markdown
* Bullet list
  * Nested bullet

1. Numbered list
2. Second item
   1. Nested numbered

- [ ] Unchecked task
- [x] Checked task
```
