Remirror provides a comprehensive collection of extensions that add specific functionality to your editor. Each extension can be composed together to create powerful editing experiences.

## What are Extensions?

Extensions are the building blocks of Remirror editors. They:
- Define document schema (nodes and marks)
- Provide commands for content manipulation
- Handle keyboard shortcuts and input rules
- Add paste handling and event listeners
- Enable collaborative editing features

## Using Extensions

```typescript
import { Remirror, useRemirror } from '@remirror/react';
import { BoldExtension, ItalicExtension, HeadingExtension } from 'remirror/extensions';

function MyEditor() {
  const { manager } = useRemirror({
    extensions: () => [
      new BoldExtension(),
      new ItalicExtension(),
      new HeadingExtension(),
    ],
  });

  return <Remirror manager={manager} />;
}
```

## Extension Options

Most extensions accept options to customize behavior:

```typescript
new HeadingExtension({
  levels: [1, 2, 3], // Only H1, H2, H3
  defaultLevel: 2,
});

new BoldExtension({
  weight: 600, // Custom font weight
});
```

## Commands

Extensions provide commands through the `commands` object:

```typescript
const { commands } = manager;

// Toggle formatting
commands.toggleBold();
commands.toggleItalic();

// Set content
commands.setHeading({ level: 1 });
commands.insertText('Hello world');
```

## Keyboard Shortcuts

Many extensions include built-in keyboard shortcuts:

- `Mod-b` - Toggle bold
- `Mod-i` - Toggle italic
- `Mod-u` - Toggle underline
- `Mod-Shift-7` - Toggle bullet list
- `Mod-Shift-8` - Toggle ordered list
- `Ctrl-Shift-1` - Heading level 1
