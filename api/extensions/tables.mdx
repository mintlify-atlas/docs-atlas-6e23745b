Table extensions provide comprehensive spreadsheet-like table functionality in your editor.

## TableExtension

Core table support with rows, cells, and advanced features.

### Installation

```typescript
import { TableExtension } from 'remirror/extensions';

const extensions = () => [new TableExtension()];
```

### Options

<ParamField path="resizable" type="boolean" default="true">
  Enable column resizing
  
  ```typescript
  new TableExtension({ resizable: true })
  ```
</ParamField>

<ParamField path="resizeableOptions" type="TableResizableOptions">
  Column resizing configuration
  
  <ParamField path="resizeableOptions.handleWidth" type="number">
    Width of resize handles in pixels
  </ParamField>
  
  <ParamField path="resizeableOptions.cellMinWidth" type="number">
    Minimum cell width in pixels
  </ParamField>
  
  ```typescript
  new TableExtension({ 
    resizeableOptions: {
      handleWidth: 5,
      cellMinWidth: 25
    }
  })
  ```
</ParamField>

<ParamField path="tabKeyboardShortcuts" type="boolean" default="false">
  Use Tab/Shift-Tab for cell navigation
  
  ```typescript
  new TableExtension({ tabKeyboardShortcuts: true })
  ```
</ParamField>

### Commands

#### `createTable(options?)`

Create a new table at the current position.

<ParamField path="options.rowsCount" type="number" default="3">
  Number of rows
</ParamField>

<ParamField path="options.columnsCount" type="number" default="3">
  Number of columns
</ParamField>

<ParamField path="options.withHeaderRow" type="boolean" default="true">
  Include header row
</ParamField>

```typescript
// Create default 3x3 table
commands.createTable();

// Create custom 5x4 table
commands.createTable({ 
  rowsCount: 5, 
  columnsCount: 4,
  withHeaderRow: true 
});
```

#### `deleteTable()`

Delete the entire table.

```typescript
commands.deleteTable();
```

## Cell Commands

#### `mergeTableCells()`

Merge selected cells.

```typescript
commands.mergeTableCells();
```

#### `splitTableCell()`

Split a merged cell.

```typescript
commands.splitTableCell();
```

#### `toggleTableCellMerge()`

Toggle between merged and split states.

```typescript
commands.toggleTableCellMerge();
```

#### `setTableCellBackground(color)`

Set cell background color.

<ParamField path="color" type="string | null" required>
  Background color (hex, rgb, or null to remove)
</ParamField>

```typescript
commands.setTableCellBackground('#FFE5E5');
commands.setTableCellBackground('rgb(255, 229, 229)');
commands.setTableCellBackground(null); // Remove background
```

#### `setTableCellAttribute(name, value)`

Set custom cell attribute.

```typescript
commands.setTableCellAttribute('colspan', 2);
commands.setTableCellAttribute('rowspan', 3);
```

## TableCellExtension

### Schema

## Usage Example

```tsx
import { Remirror, useRemirror } from '@remirror/react';
import { TableExtension } from 'remirror/extensions';

function TableEditor() {
  const { manager } = useRemirror({
    extensions: () => [
      new TableExtension({
        resizable: true,
        tabKeyboardShortcuts: true,
        resizeableOptions: {
          cellMinWidth: 50,
        },
      }),
    ],
  });

  const createTable = () => {
    manager.commands.createTable({
      rowsCount: 4,
      columnsCount: 3,
      withHeaderRow: true,
    });
  };

  return (
    <div>
      <div className="toolbar">
        <button onClick={createTable}>Create Table</button>
        <button onClick={() => manager.commands.addTableRowBefore()}>
          Add Row Above
        </button>
        <button onClick={() => manager.commands.addTableRowAfter()}>
          Add Row Below
        </button>
        <button onClick={() => manager.commands.addTableColumnBefore()}>
          Add Column Left
        </button>
        <button onClick={() => manager.commands.addTableColumnAfter()}>
          Add Column Right
        </button>
        <button onClick={() => manager.commands.deleteTableRow()}>
          Delete Row
        </button>
        <button onClick={() => manager.commands.deleteTableColumn()}>
          Delete Column
        </button>
        <button onClick={() => manager.commands.mergeTableCells()}>
          Merge Cells
        </button>
        <button onClick={() => manager.commands.splitTableCell()}>
          Split Cell
        </button>
        <button onClick={() => manager.commands.deleteTable()}>
          Delete Table
        </button>
      </div>
      <Remirror manager={manager} />
    </div>
  );
}
```

## Advanced Features

### Cell Merging

Merge cells by selecting multiple cells and using `mergeTableCells()`:

```typescript
// Select cells then merge
commands.mergeTableCells();

// Split merged cell
commands.splitTableCell();
```

### Column Resizing

When `resizable: true`, users can drag column borders to resize:

```typescript
new TableExtension({
  resizable: true,
  resizeableOptions: {
    handleWidth: 5,        // Resize handle width
    cellMinWidth: 25,      // Minimum column width
  },
});
```

### Cell Backgrounds

Apply background colors to cells:

```typescript
// Set background
commands.setTableCellBackground('#FFE5E5');

// Remove background
commands.setTableCellBackground(null);
```

### Header Rows/Columns

Designate rows or columns as headers:

```typescript
// Toggle first row as header
commands.toggleTableHeaderRow();

// Toggle first column as header
commands.toggleTableHeaderColumn();

// Check if has header
if (helpers.tableHasHeader('row')) {
  // Table has header row
}
```

## React Tables Extension

For advanced React-specific table features:

```typescript
import { ReactTablesExtension } from 'remirror/extensions';

const extensions = () => [
  new ReactTablesExtension(),
];
```

Provides React components for table controls and cell editing.
