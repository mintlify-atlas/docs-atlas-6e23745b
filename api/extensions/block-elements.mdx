Block element extensions define the structural nodes in your editor's document.

## HeadingExtension

Adds heading nodes (H1-H6) to your editor.

### Installation

```typescript
import { HeadingExtension } from 'remirror/extensions';

const extensions = () => [new HeadingExtension()];
```

### Options

<ParamField path="levels" type="number[]" default="[1, 2, 3, 4, 5, 6]">
  Supported heading levels
  
  ```typescript
  new HeadingExtension({ levels: [1, 2, 3] })
  ```
</ParamField>

<ParamField path="defaultLevel" type="number" default="1">
  Default heading level when toggling
  
  ```typescript
  new HeadingExtension({ defaultLevel: 2 })
  ```
</ParamField>

### Commands

#### `toggleHeading(attrs?)`

Toggle heading at the current block.

<ParamField path="attrs.level" type="number">
  Heading level (1-6)
</ParamField>

```typescript
// Toggle to default level
commands.toggleHeading();

// Toggle to specific level
commands.toggleHeading({ level: 2 });
```

### Keyboard Shortcuts

- `Ctrl-Shift-1` - Heading level 1
- `Ctrl-Shift-2` - Heading level 2
- `Ctrl-Shift-3` - Heading level 3
- `Ctrl-Shift-4` - Heading level 4
- `Ctrl-Shift-5` - Heading level 5
- `Ctrl-Shift-6` - Heading level 6

### Input Rules

Markdown-style patterns:
- `# ` - Creates H1
- `## ` - Creates H2
- `### ` - Creates H3
- `#### ` - Creates H4
- `##### ` - Creates H5
- `###### ` - Creates H6

### Paste Rules

Automatically converts pasted markdown headings:
- `# Heading` - Converts to H1
- `## Heading` - Converts to H2

## ParagraphExtension

The fundamental text block node. Required for most editors.

### Installation

```typescript
import { ParagraphExtension } from 'remirror/extensions';

const extensions = () => [new ParagraphExtension()];
```

### Commands

#### `convertParagraph(options?)`

Convert current node to a paragraph.

<ParamField path="options.attrs" type="ProsemirrorAttributes">
  Attributes to apply
</ParamField>

<ParamField path="options.selection" type="PrimitiveSelection">
  Selection range
</ParamField>

<ParamField path="options.preserveAttrs" type="boolean">
  Preserve existing attributes
</ParamField>

```typescript
commands.convertParagraph();
commands.convertParagraph({ attrs: { class: 'custom' } });
```

#### `insertParagraph(content, options?)`

Insert a new paragraph with content.

<ParamField path="content" type="string" required>
  Text content for the paragraph
</ParamField>

```typescript
commands.insertParagraph('Hello world');
commands.insertParagraph('New paragraph', { 
  selection: { from: 10 } 
});
```

### Keyboard Shortcuts

- `Ctrl-Shift-0` - Convert to paragraph

## Usage Example

```tsx
import { Remirror, useRemirror } from '@remirror/react';
import {
  DocExtension,
  TextExtension,
  ParagraphExtension,
  HeadingExtension,
  BlockquoteExtension,
  HorizontalRuleExtension,
  HardBreakExtension,
} from 'remirror/extensions';

function BlockEditor() {
  const { manager } = useRemirror({
    extensions: () => [
      new DocExtension(),
      new TextExtension(),
      new ParagraphExtension(),
      new HeadingExtension({ levels: [1, 2, 3] }),
      new BlockquoteExtension(),
      new HorizontalRuleExtension(),
      new HardBreakExtension(),
    ],
  });

  return (
    <div>
      <div className="toolbar">
        <button onClick={() => manager.commands.toggleHeading({ level: 1 })}>
          H1
        </button>
        <button onClick={() => manager.commands.toggleHeading({ level: 2 })}>
          H2
        </button>
        <button onClick={() => manager.commands.toggleBlockquote()}>
          Quote
        </button>
        <button onClick={() => manager.commands.insertHorizontalRule()}>
          HR
        </button>
      </div>
      <Remirror manager={manager} />
    </div>
  );
}
```

## Schema Structure

```typescript
// Document structure
doc {
  block+ // One or more blocks
}

heading(level: 1-6) {
  inline* // Inline content
}

paragraph {
  inline* // Inline content
}

blockquote {
  block+ // Nested blocks
}

horizontalRule // Empty node

hardBreak // Empty node
```
